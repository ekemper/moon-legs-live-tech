# Moon Legs Live Tech

Live performance tools and utilities for electronic music production.

## Projects

### [ddti-sequencer](./ddti-sequencer/)

MIDI note sequencer for DDrum DDTI drum triggers → Ableton Live. Converts drum hits into cycling note sequences for creating basslines and melodies.

Each trigger hit advances to the next note in your custom sequence, turning your drum brain into a melodic sequencer.
